# py-pratice_poo

**KATA I:** Evaluacion de Alumnos
 Vamos a desarrollar un programa donde nos facilite la evaluación de los alumnos. Para ello vamos a usar un POO.

Alumno:
    Propiedades:
    - Nombre
    - Apellidos
    - DNI
    - Edad
    - Nota

    Metodos:
    - Saludar
    - Añadir nota (0-10)
    - Cumplir años (+1 a la edad)
